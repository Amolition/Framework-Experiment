Orthogonality

Given the vectors $ \vec{V}_{n}$ and $\vec{V}_{m},$ we define their inner product as $$\vec{V}_{n} \cdot \vec{V}_{m} = \sum_{i}V_{n}^{(i)} V_{m}^{(i)}, $$ where $V_{n}^{(i)}$ is the $i^{\text{th}}$ component of the vector $\vec{V}_{n}$. If $$ \vec{V}_{n} \cdot \vec{V}_{m} = 0 \text{ for } m \neq n,$$ then we say the vectors are orthogonal. If you move in the direction of any vector $\vec{a}$, you do not change the position component in the direction of any vector orthogonal to $\vec{a}$. In other words, orthogonal vectors are perpendicular.

If $ \vec{V}_{n} \cdot \vec{V}_{m} = \delta_{nm}$ then the vectors are said to be orthonormal. $$\delta_{nm} = \left\{\begin{matrix}0 \quad \text{if } m\neq n \\ 1 \quad \text{if } m = n \end{matrix}\right. $$ This mean the vectors are orthogonal and conveniently normalised.

Use the visualisation to see that when the basis vectors are not parallel they span $\mathbb{R}^{2}$, using these as basis vectors, we can construct any other vector in $\mathbb{R}^{2}$.

If you can represent any vector in $\mathbb{R}^{n}$ using $n$ vectors, then the basis vectors form a complete set. If the set of these vectors are all orthonormal, then we have a complete orthonormal set.

In the same way as before, we will now demonstrate what it means for functions to be complete. We define the functions $f_{n},f_{m}$ to exist on the domain $\left[a,b \right]$.

We define their inner product to be $$\langle f_{n} , f_{m} \rangle = \int_{a}^{b} f_{n}\left(x \right)f_{m}^{*}\left(x \right)dx$$ and say the functions are orthogonal if $\langle f_{n} , f_{m} \rangle = 0 $ for $m \neq n$. We then say the functions are orthonormal if $\langle f_{n} , f_{m} \rangle = \delta_{nm} $.

We now claim that you can construct any square integrable function from a complete set of basis functions in the same way vectors could be constructed from a complete set of basis vectors. A square integrable function $f$ is a function whose inner product with itself, $ \langle f,f \rangle $ is finite.

Introduction to Fourier series

Consider a function $f(x)$ which is periodic over the domain $ -L \leq x < L $ where $L$ is positive. $f(x)$ can be represented as a sum of sines and cosines with varying frequencies. These trigonometric functions form orthogonal basis.

The Fourier series of $f(x)$ is

$\displaystyle f(x) = \frac{a_0}{2}$ $+ \displaystyle\sum_{n=1}^{\infty} \Big[ a_n\cos\Big(\frac{\pi nx}{L}\Big)$ $\displaystyle+ b_n\sin\Big(\frac{\pi nx}{L}\Big) \Big]$ $ \qquad \left[ 1 \right]$

where

$\displaystyle a_n = \displaystyle\frac{1}{L}\int_{-L}^{L} f(x)$ $\displaystyle \cos\Big(\frac{\pi nx}{L}\Big) dx, $

$\displaystyle b_n = \displaystyle\frac{1}{L}\int_{-L}^{L} f(x)$ $\displaystyle \sin\Big(\frac{\pi nx}{L}\Big) dx, $

$\displaystyle a_0 = \displaystyle\frac{1}{L}\int_{-L}^{L} f(x) dx, $

and

$$ n=0,1,2...$$

Note: $a_0$$/2$ represents the average value of $f(x)$.

The Fourier series representation of the function $f(x)$ in exponential (complex) form is

$\displaystyle f(x) =\sum_{n=-\infty}^{\infty} c_n e^{{in \pi x}/L}$ $\displaystyle \qquad \left[ 2 \right]$

where

$\displaystyle c_n = \frac{1}{2L}\int_{-L}^{L} f(x)$ $\displaystyle e^{{-in \pi x}/L} dx, $

Note: The constants in front of the integral(s) of the coefficient for the trigonometric and exponential fourier series are different. Also note the limit of the summations.

Derivation of the Fourier Series

We begin by assuming that $f(x)$ can be represented as a sum of exponential terms.

$\displaystyle f(x) = \displaystyle\sum_{n=-\infty}^{\infty} c_n e^{{in \pi x}/L}$

Multiplying by $e^{{-im \pi x}/L}$, where $m=0,1,2...$, and integrating both sides gives the following:

$\displaystyle\int_{-L}^{L} f(x) e^{{-im \pi x}/L} dx$ $=\displaystyle\sum_{n=-\infty}^{\infty} c_n $ $\displaystyle \int_{-L}^{L} e^{{i(n-m) \pi x}/L} dx. $

Performing the integral on the R.H.S gives

$ \displaystyle\int_{-L}^L e^{(i\pi x/L)(n-m)}dx$ $\displaystyle =\left\{ \begin{array}{ll} 2L \qquad \text{(if n=m)} \\ 0 \qquad \quad \text{(if nâ‰ m)} \end{array} \right \} $ $\displaystyle= 2L \cdot \delta_{nm}.$

This is the orthonormality of complex exponentials.

Now since the only non-zero value of the sum is when $n=m$,

$\displaystyle\int_{-L}^L f(x) e^{-im\pi x/L}dx$ $\displaystyle= 2L \sum_{n=-\infty}^{+\infty}c_n\delta_{mn}$ $\displaystyle=2L\cdot c_m, $

hence

$\displaystyle c_m=\frac{1}{2L}\int_{-L}^{+L}f(x) $ $\displaystyle e^{-im\pi x/L}dx. $

Note: As $m$ is an arbitrary integer, it can be assigned as $n$ for simplicity.

Given the previously derived equation, we can derive the trigonometric form of the Fourier series. By reorganising the sum, we get

$\displaystyle f(x)= \underbrace{c_0}_\text{n=0}$ $\displaystyle + \sum_{n=1}^{\infty} \big( \underbrace{c_n e^{in\pi x/L}}_\text{Positive n term}$ $\displaystyle + \underbrace{c_{-n} e^{-in\pi x/L}}_\text{Negative n terms} \big).$

We can regroup the terms in the summation in the following way:

$ \displaystyle = \frac{1}{2} \Big[ 2c_n e^{in\pi x/L}$ $\displaystyle + 2c_{-n} e^{-in\pi x/L}$ $ \displaystyle + \underbrace{c_{-n} e^{in\pi x/L} - c_{-n} e^{in\pi x/L}}_\text{ =0}$ $\displaystyle + \underbrace{c_{n} e^{-in\pi x/L} -c_{n} e^{-in\pi x/L}}_\text{ =0} \Big] $

$ \displaystyle = \frac{1}{2} \Big[ c_n e^{in\pi x/L} $ $\displaystyle +c_{-n} e^{in\pi x/L}$ $ \displaystyle + c_n e^{-in\pi x/L} $ $\displaystyle + c_{-n} e^{-in\pi x/L}$ $ \displaystyle + c_n e^{in\pi x/L} $ $\displaystyle - c_{-n} e^{in\pi x/L}$ $ \displaystyle - c_n e^{-in\pi x/L} $ $\displaystyle +c_{-n} e^{-in\pi x/L}\Big] $

$\displaystyle f(x)= c_0 + \sum_{n=1}^{\infty}$ $\displaystyle \Big( \frac{c_n + c_{-n}}{2} \big(e^{in\pi x/L}$ $\displaystyle +e^{-in\pi x/L} \big)$ $+ \displaystyle \frac{c_n - c_{-n}}{2} \big(e^{in\pi x/L} $ $\displaystyle -e^{-in\pi x/L} \big) \Big) $

Now define $ a_n = c_n + c_{-n}$ and $ b_n = i(c_n - c_{-n})$ and use that these exponentials can we written as sines and cosines.

Note: $a_0 = 2c_0$ and $b_0=0$.

Hence

$ f(x) = \displaystyle\frac{a_0}{2}$ $+ \displaystyle\sum_{n=1}^{\infty} \Big[ a_n\cos\Big(\frac{\pi nx}{L}\Big)$ $+ \displaystyle b_n\sin\Big(\frac{\pi nx}{L}\Big) \Big]$

$a_n$ can be derived by using the expression from $c_n$

$\displaystyle a_n = c_n + c_{-n}$ $\displaystyle =\frac{1}{2L}\int_{-L}^{+L}f(x)$ $\displaystyle e^{-in\pi x/L}dx$ $\displaystyle +\frac{1}{2L}\int_{-L}^{+L}f(x)$ $\displaystyle e^{in\pi x/L}dx$

Factorise and spot the exponential form of $\cos$.

$\displaystyle =\frac{1}{L}\int_{-L}^{+L}f(x)$ $\displaystyle \Big[\frac{e^{in\pi x/L} +e^{-in\pi x/L}}{2}\Big]dx$ $\displaystyle= \displaystyle\frac{1}{L}\int_{-L}^{L} f(x)$ $\displaystyle \cos\Big(\frac{\pi nx}{L}\Big) dx$

Similarly for $b_n$,

$ b_n = \displaystyle\frac{1}{L}\int_{-L}^{L} f(x)$ $\displaystyle \sin\Big(\frac{\pi nx}{L}\Big) dx. $

Triangular Fourier Series Derivation

The triangular function is defined by the following:

$\displaystyle f(x)=\left\{ \begin{array}{ll} \frac{-2A}{L} (x+L) \quad (-L \leq x < -L/2) \\ \frac{2A}{L} x \qquad \qquad (-L/2\leq x < L/2) \\ \frac{-2A}{L} (x-L) \quad (L/2\leq x < L) \end{array} \right. $

First notice that $f(x)$ is odd about $x=0$, hence $a_n=0$ for all $n$. Also, since the $b_n$ integrand is even about $x=0$,

$\displaystyle b_n=\frac{2}{L}\int_0^L f(x)$ $\displaystyle \sin \left(\frac{n \pi x}{L}\right)dx. $

Notice that for even $n$, $b_n$ is odd about $x=L/2$. Hence,

$ \displaystyle b_n=\frac{2}{L}\int_0^{L} f(x)$ $\displaystyle \sin \left(\frac{n\pi x}{L} \right) dx$ $\displaystyle = 0 $ $\displaystyle\quad \text{for even n}. $

The integral from $0$ to $L/2$ is equal in magnitude but opposite in sign to the integral from $L/2$ to $L$, for even $n$.

For odd $n$, the function is even around $L/2$, so we only have to integrate from $0$ to $L/2$ then multiply by $2$.This has made our job much easier.

$\displaystyle b_n=2\frac{2}{L}\int_0^{L/2} f(x)$ $\displaystyle \sin \left(\frac{n\pi x}{L} \right) dx$ $\displaystyle \quad \text{for odd n} $

$ \displaystyle \text{For 0 < x < L/2:}$ $\displaystyle \quad f(x)=\frac{2Ax}{L} $

Remembering that for even $n$, $b_n=0$, therefore only consider odd $n$.

$\displaystyle b_n = \frac{4}{L} \frac{2A}{L}\int_0^{L/2} x $ $\displaystyle \sin \left(\frac{n\pi x}{L} \right) dx $

$\displaystyle u=x \qquad $ $\displaystyle dv= \sin \left(\frac{n\pi x}{L} \right)dx $

$\displaystyle du=dx \qquad $ $\displaystyle v=\frac{-L}{n\pi} \cos \left(\frac{n\pi x}{L} \right) $

$\displaystyle b_n=\frac{8A}{L^2}$ $\displaystyle \Bigg[ \Big[ \frac{-xL}{n\pi}$ $\displaystyle \cos \Big(\frac{n\pi x}{L}\Big) \Big] \Bigr|_0^{L/2} $ $\displaystyle -\int_0^{L/2}\frac{-L}{n\pi}$ $\displaystyle \cos \Big(\frac{n\pi x}{L} \Big)dx \Bigg] $

For odd n,

$\displaystyle \cos \left(\frac{n\pi}{2} \right)=0 $

Therefore,

$\displaystyle b_n=\frac{8A}{L^2} \Bigg[0+ $ $\displaystyle \int_0^{L/2}\frac{L}{n\pi}$ $\displaystyle \cos \left(\frac{n\pi x}{L} \right)dx \Bigg]$

$\displaystyle b_n=\frac{8A}{L^2} \frac{L^2}{(n\pi)^2}$ $\displaystyle \sin \left(\frac{n\pi x}{L} \right) \biggr |_0^{L/2} $

Note for odd $n$ :

$ \displaystyle \sin \left(\frac{n\pi}{2} \right)=(-1)^{\frac{n-1}{2}} $

Therefore,

$ \displaystyle b_n=\frac{8A}{n^2\pi^2}(-1)^{\frac{n-1}{2}} $

Finally,

$\displaystyle f(x)=\sum_{n=1,3,5...}^\infty \frac{8A}{n^2\pi^2}$ $\displaystyle(-1)^{\frac{n-1}{2}} $ $\displaystyle \sin \left(\frac{n\pi x}{L} \right) $

And the using that an odd number is given by $2n +1$ for any $n$,

$ \displaystyle f(x)=\sum_{n=1,2,3...}^\infty $ $\displaystyle \frac{8A}{(2n-1)^2\pi^2}(-1)^n $ $\displaystyle \sin \left(\frac{(2n-1)\pi x}{L} \right) $

Parabolic Fourier Series Derivation

The parabolic function is defined as:

$ \displaystyle f(x)=Ax^2 \quad \text{for:} -L\leq x< L$

and forced to be periodic. $A$ is a constant.

Therefore, $a_n$ and $b_n$ coefficients are given as

$\displaystyle b_n = \frac{1}{L}\int_{-L}^{+L} Ax^2$ $\displaystyle \sin \left(\frac{n \pi x}{L} \right)dx $

$\displaystyle a_n= \frac{1}{L}\int_{-L}^{+L} Ax^2$ $\displaystyle \cos \left(\frac{n \pi x}{L} \right)dx $

The $b_n$ integral is odd about $x=0$, hence $b_n=0$ for all $n$. $f(x)$ is even about $x=0$. Hence

$\displaystyle a_n=\frac{2}{L}\int_{0}^{+L} Ax^2$ $\displaystyle \cos \left(\frac{n \pi x}{L} \right)dx $

For $n=0$:

$\displaystyle a_0 = \frac{2}{L} \int _{0}^{+L} Ax^2 dx $ $\displaystyle a_0 = \frac{2AL^2}{3} $

For $n\neq 0 $:

$\displaystyle a_n = \frac{2A}{L}\int_0^L x^2$ $\displaystyle \cos \left(\frac{n\pi x}{L} \right)dx $

$\displaystyle du = 2x\cdot dx \qquad$ $\displaystyle v=\frac{L}{n\pi} \sin \left(\frac{n\pi x}{L} \right) $

$\displaystyle u=x^2 \qquad$ $\displaystyle dv= \cos \left(\frac{n \pi x}{L} \right)dx $

$\displaystyle a_n = \frac{2A}{L}$ $\displaystyle \Bigg[ \Big[\frac{Lx^2}{n\pi}$ $\displaystyle \sin \left(\frac{n\pi x}{L} \right) \Big] \biggr |_0^L $ $\displaystyle -\int_0^L\frac{2xL}{n\pi}$ $\displaystyle \sin \left(\frac{n \pi x}{L} \right)dx \Bigg] $

$\displaystyle = \frac{2A}{L}$ $\displaystyle \Bigg[0$ $\displaystyle -\int_0^L\frac{2xL}{n\pi}$ $\displaystyle \sin\left(\frac{n \pi x}{L}\right)dx \Bigg] $

$\displaystyle a_n = -\frac{4A}{n\pi}\int_{0}^{L} x$ $\displaystyle \sin \left(\frac{n \pi x}{L} \right)dx $

$\displaystyle u = x \qquad$ $\displaystyle dv = \sin \left(\frac{n\pi x}{L} \right) dx$

$\displaystyle du = dx \qquad$ $\displaystyle v = -\frac{L}{n\pi}$ $\displaystyle \cos \left(\frac{n\pi x}{L} \right) $

$\displaystyle a_n = \frac{-4A}{n\pi}$ $\displaystyle \Bigg[\Big[-\frac{xL}{n\pi}$ $\displaystyle \cos \left(\frac{n\pi x}{L} \right)\Big] \biggr |_0^L $ $\displaystyle - \int_0^L\frac{-L}{n\pi} $ $\displaystyle \cos \left(\frac{n\pi x}{L} \right)dx \Bigg] $

Dirac-Delta Fourier Series Derivation

The Dirac-delta function can be thought of as a infinite spike at $x=0$.

$\displaystyle f(x) = \delta(x)=\left\{ \begin{array}{ll} +\infty \quad (x=0) \\ 0 \qquad (x \neq 0) \end{array} \right. $

$\displaystyle a_n=\frac{1}{L}\int_{-L}^{+L} \delta (x)$ $\displaystyle \cos \left(\frac{n\pi x}{L} \right)dx $

$\displaystyle b_n=\frac{1}{L}\int_{-L}^{+L}\delta(x)$ $\displaystyle \sin \left(\frac{n\pi x}{L} \right)dx $

$\delta(x)$ is an even function. Hence $b_n=0 $ and

$\displaystyle a_n=\frac{1}{L} $

By considering the shifting property of the Dirac-Delta function,

$\displaystyle \int_{\alpha-\epsilon}^{\alpha+\epsilon} g(x) \delta(x-\alpha)dx=g(\alpha)$

For us, $\alpha=0, \epsilon=L,$ and $g(x)= \sin \left(n \pi x/L \right).$

Therfore,

$\displaystyle a_n=\frac{1}{L}\cos \left(\frac{n\pi \times 0}{L} \right)=\frac{1}{L}$

$\displaystyle b_n=\frac{1}{L}\sin \left(\frac{n\pi \times 0}{L} \right)=0$

Therefore, the fourier series for positive $L$ can be written as

$\displaystyle f(x)=\frac{1}{2L}+\sum_{n=1}^\infty \frac{1}{L}$ $\displaystyle \cos \left(\frac{n\pi x}{L} \right). $

Square Wave Derivation

The square wave is defined as the following:

$\displaystyle f(x)=\left\{ \begin{array}{ll} -A \quad (-L \leq x < 0) \\ A \quad (0\leq x < L) \end{array} \right. $

The $a_{n}$ terms can be found be noting that $f(x)$ is odd, so $a_n$ is $0$ for all $n$.

To find the $b_n$ terms, the integration can be split into two parts: $-L\leq x <0$ and $ 0\leq x < L.$ This gives the following integration:

$\displaystyle b_n=\frac{1}{L}\int_{-L}^0 -A$ $\displaystyle \sin \left(\frac{n\pi x}{L} \right)dx$ $\displaystyle + \frac{1}{L}\int_0^L A$ $\displaystyle \sin \left(\frac{n \pi x}{L} \right)dx $

which can be evaluated to:

$\displaystyle b_n=\frac{-2A}{L}\frac{L}{n\pi}$ $\displaystyle \cos \left(\frac{n\pi x}{L} \right)\biggr |_0^L$ $\displaystyle =\frac{2A}{n\pi}\left[1-\cos\left(n\pi \right) \right] $

Note that $\displaystyle\cos \left(n \pi\right)=(-1)^n$

Hence $\displaystyle b_n=\frac{2A}{n\pi}(1-(-1)^n)$

Therefore the final fourier series is:

$\displaystyle f(x)=\sum_{n=1}^{\infty} \frac{2A}{n\pi}$ $\displaystyle (1-(-1)^n)$ $\displaystyle\sin \left(\frac{n\pi x}{L}\right). $

Sawtooth Wave Derivation

The Sawtooth wave is defined as the following:

$\displaystyle f(x)=\frac{Ax}{L} $ $\displaystyle \quad (-L \leq x < L). $

$f(x)$ is odd, so $a_n= 0$ for all $n$.

The $b_n$ terms are given by the following integration:

$\displaystyle b_n=\frac{1}{L}\int_{-L}^{L} \frac{Ax}{L}$ $\displaystyle\sin \left(\frac{n\pi x}{L} \right)dx $ $\displaystyle =\frac{2A}{L^{2}}\int_{0}^{L} x$ $\displaystyle \sin \left(\frac{n\pi x}{L} \right)dx. $

Using Integration by parts it can be shown that:

$\displaystyle u=x \quad $ $\displaystyle dv=\sin \left(\frac{n\pi x}{L} \right)dx $

$\displaystyle du=dx \quad$ $\displaystyle v=\frac{-L}{n\pi}\cos \left(\frac{n\pi x}{L} \right) $

$\displaystyle b_n=\frac{2A}{L^{2}} \Bigg[ \frac{-Lx}{n\pi}$ $\displaystyle \cos \left(\frac{n\pi x}{L} \right)\biggr |_0^L $ $\displaystyle-\int_0^L\frac{-L}{n\pi}$ $\displaystyle \cos \left(\frac{n\pi x}{L} \right)dx \Bigg] $

Note that the second integrand is zero as it is of cosine over a full period.

$\displaystyle b_n=-\frac{2A}{n\pi} \cos \left(n\pi \right)$ $\displaystyle=-\frac{2A}{n\pi}(-1)^n $

$\displaystyle b_n=\frac{2A}{n\pi}(-1)^{n+1}. $

Therefore the final fourier series is:

$\displaystyle f(x)=\sum_{n=1}^\infty \frac{2A}{n\pi}(-1)^{n+1}$ $\displaystyle \sin \left(\frac{n\pi x}{L} \right). $

By integrating by parts,

$\displaystyle u=x \qquad$ $\displaystyle dv= \cos\left(\frac{n\pi x}{L}\right)dx$

$\displaystyle du=dx \qquad$ $\displaystyle v=\frac{L}{n\pi} \sin \left(\frac{n\pi x}{L} \right) $

$\displaystyle a_n = \frac{2A}{L}$ $\displaystyle \Bigg[ \Big[\frac{Lx}{n\pi}$ $\displaystyle \sin \left( \frac{n \pi x}{L} \right) \Big] \Biggr|_0^L$ $\displaystyle -\int_0^{+L} \frac{L}{n\pi}$ $\displaystyle \sin \left(\frac{n \pi x}{L} \right)dx \Bigg]$ $\displaystyle = \frac{2A}{L} \Bigg[ 0 $ $\displaystyle -\int_0^{+L}\frac{L}{n\pi}$ $\displaystyle \sin \left(\frac{n\pi x}{L} \right)dx \Bigg]$

$\displaystyle a_n = \frac{2A}{L}\frac{L^2}{(n\pi)^2}$ $\displaystyle \cos \left( \frac{n\pi x}{L} \right)\biggr |_0^L . $

Therefore the final fourier series is:

$\displaystyle f(x)=\frac{AL}{2}+$ $\displaystyle \sum_{n=1}^\infty \frac{2AL}{(n\pi)^2}$ $\displaystyle \left[(-1)^n-1 \right]$ $\displaystyle \cos \left(\frac{n\pi x}{L} \right). $

Section 5

Power spectrum

The power spectrum form is a different representation of a continuous signal. One can think of a signal as a sound wave that is a superposition of different frequencies $\omega_n$, each with amplitude $\alpha_n$, and relative phases $\theta_n$. Any signal of period $2L$ can be decomposed in terms of the amplitude and relative phase of its discrete frequency modes. Its two parameters $\alpha_n$ and $\theta_n$, enable us to measure the amount of each frequency $\omega_n=\frac{n\pi}{L}$ contained in the signal, and the relative phase difference between the $n^{th}$ frequency and a cosine wave centered at $x=0$. This is simply a different way to represent a signal rather than decomposing it into its even and odd parts.

$$ f(x) = \frac{a_0}{2} + \sum_{n=1}^{\infty}\alpha_n \cos\bigg(\frac{n\pi x}{L} - \theta_n\bigg)$$ $$ \alpha_n = \sqrt{a^2_n + b^2_n}$$ $$\theta_n = \tan^{-1} \bigg(\frac{b_n}{a_n}\bigg)$$

For some even functions, when the period transitions from 0 to negative, the value of $\theta_n$ change from $0$ to $\pi$ or vice versa. Why do you think this happens? (This happens because the value of $a_n$ changes sign ($b_n$ is 0) causing the value of $\theta_n$ to change. For $b_n = 0$, $\tan^{-1}(\frac{b_n}{a_n})$ can be both 0 or $\pi$ depending on whether $a_n$ is positive or negative. Not all even functions exhibit this. The parabola for example, has its $a_n$ proportional to $L^2$, so its sign does not change when L changes sign.

Orthogonal functions

Two functions $f$ and $g$ are said to be orthogonal if their inner product $\langle f,g\rangle$ is 0. Just like basis vectors in a finite vector space, a set of orthogonal functions can form an infinite basis in a function space and a linear sum of these basis functions can describe any function in that function space. For functions that are periodic over $-L < x \leq L$ with a period of $2L$, the fourier components $\rightarrow$ $\sin(\frac{\pi x}{L}), \cos(\frac{\pi x}{L}), \sin(\frac{2\pi x}{L}), \cos(\frac{2\pi x}{L})...$ and so on form a complete orthogonal set that can be used to describe these functions. This means that increasing the amplitude of one of the components does not affect the other components, so changing the amount of any frequency contained in a signal leaves all the other frequencies unaffected.

To specify a custom function, select custom from the drop down menu and input your desired function into the input for $f(x)$.

The following functions can be used:

  • Trigonometric functions (e.g. sin(), cos())
  • Trigonometric inverses (e.g. arcsin(), arccos())
  • Hyperbolic functions (e.g. sinh(), cosh())
  • Hyperbolic inverses (e.g. arsinh(), arcosh())
  • Exponentials and logs (e.g. exp(), log())

Functions should be real and fully defined over the domain, for example log(x) is imaginary for values less than zero, so does not work. log(x+1) does not work either (with a period of 1), since log is undefined at zero, however log(1+x^2) does work for all periods, since it is defined over the whole domain and is real.

Sometimes due to the numerical integration methods used, the power spectrum shows very small values ($10^{-17}$). These are simply due to the inexact integration and should be treated as zero.
